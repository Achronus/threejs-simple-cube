# Threejs Simple Cube
A repository dedicated to my first Threejs project that explores the fundamentals.

It uses Vite + Vanilla JS.

### Installation
Once you've cloned or downloaded this project file to your local machine, navigate to this project directory in your terminal.

Run the following command to install the necessary dependencies:

``` bash
npm install
```
### Running the project
To start the development server, run the following command:

``` bash
npm run dev
```
This will start the server and open your default browser to your localhost. The site will reload automatically as you make changes to your code.
